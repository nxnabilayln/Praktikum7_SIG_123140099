## 📖 Deskripsi

Praktikum ini bertujuan untuk membangun REST API yang dapat mengakses dan mengelola data spasial menggunakan FastAPI dan PostgreSQL dengan ekstensi PostGIS.

API yang dibuat mampu melakukan operasi CRUD serta query spasial seperti pencarian lokasi terdekat (nearby). Selain itu, API juga mendukung output dalam format GeoJSON untuk integrasi dengan sistem pemetaan.
